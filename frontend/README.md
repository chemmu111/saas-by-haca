# Frontend - Social Media Management Platform

## Structure

```
frontend/
└── public/
    ├── app.js
    ├── home.html
    ├── login.html
    ├── signup.html
    └── styles.css
```

## Files

- **app.js** - Main frontend JavaScript with authentication logic
- **login.html** - Login page
- **signup.html** - Signup page
- **home.html** - Home page (protected)
- **styles.css** - Stylesheet

## API Integration

The frontend communicates with the backend API at `/api/auth/login` and `/api/auth/signup`.
